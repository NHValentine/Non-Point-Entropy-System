##### Executions notes...



All of these tests are configured for 32 threads and maximum CPU utilization.  They'll reproduce with minimal resources, but running them as configured will push your system hard. Make sure your system can handle sustained CPU loads and watch your temps! ⚠️The base 10 and -2 test runs for nearly an hour at max usage.⚠️ I've reproduced the results with two threads at ~10% usage while I watched youtube though.  So, you dont HAVE to push your system to see the results.



Proceed accordingly. :D
