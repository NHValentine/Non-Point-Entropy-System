##### Naming Convention



All of the tests use a pretty simple naming system so you can tell at a glance what they are.



NPES\_\[bitspace]\_\[base/s]



Roman numerals are used for base notation

UPPERCASE = positive base

lowercase = negative base

multiple bases are seperated by \_ underscores

multiples of the same are written as exponents



##### Examples



NPES\_2048\_x  -> 2048 bit space, base neg 10

NPES\_256\_X\_iii -> 256 bit space, bases pos 10 and neg 3

NPES\_256\_iv^2 -> 256 bit space, two instances of base neg 4
